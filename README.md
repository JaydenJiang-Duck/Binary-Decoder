# Binary-Decoder
A small binary decoder that is controlled by the user. The user can switch on and off 4 switches, essentially creating a 4-digit number that can be decoded into a number.

Here is how the code works:
1) The user flips the switches to create a 4-digit series of numbers (eg: 0110)
2) The LCD screen displays the decoded version by multiplying each digit by a value based on where
3) Done!

The LEDs are just another way to express the digits.
